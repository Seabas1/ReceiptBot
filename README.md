# ReceiptBot_TI

Телеграм-бот для распознавания российских кассовых чеков с записью данных в Excel.

## Этапы разработки

1. Структура проекта и конфигурация
2. Генерация синтетических данных (trdg)
3. Дообучение OCR-модели (EasyOCR CRNN)
4. Предобработка изображений (OpenCV)
5. Парсинг полей чека
6. Telegram-бот и запись в Excel
7. Интеграция и тестирование

## Стек

- OCR: EasyOCR (CRNN: CNN + BiLSTM + CTC)
- Предобработка: OpenCV
- Парсинг: регулярные выражения
- Синтетические данные: trdg
- Бот: python-telegram-bot
- Excel: openpyxl

## Структура проекта

- `bot/` — Telegram-бот и запись в Excel
- `processing/` — предобработка изображений, OCR, парсинг чека
- `model/` — генерация данных, обучение, оценка модели
- `data/` — датасеты (samples, synthetic, real)
- `config.py`, `main.py`, `requirements.txt`
